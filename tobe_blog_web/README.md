## Prerequisites

- Git
- NodeJS
- NPM

### Tips

本项目默认使用volta来管理node版本，如果你没有安装volta，可以使用nvm来管理node版本，但是需要注意node版本的兼容性问题。

> 什么是volta? [volta](https://docs.volta.sh/guide/getting-started)

## Install dependencies

```bash
npm i
```

## How to run

```bash
npm run start
```

## How to build

```bash
npm run build:prod
```
